# 🏟️ Sports Zone – Sports Hall Booking Portal

## Project Overview

Sports Zone is a ServiceNow Service Portal application developed to simplify sports hall booking. Users can browse available venues, submit booking requests, view booking history, and receive booking confirmation through an easy-to-use portal.

---

## Features

- Hero Banner
- Sports Hall Navigation
- Popular Venues
- Sports Hall Booking Form
- Booking Success Page
- My Bookings Page
- Help & Support Page
- Backend Booking Management

---

## Technologies Used

- ServiceNow Service Portal
- Widgets
- HTML
- CSS
- AngularJS
- ServiceNow Forms
- ServiceNow Tables

---

## Project Workflow

1. User logs into ServiceNow.
2. Opens the Sports Zone Portal.
3. Browses available sports venues.
4. Clicks Book Now.
5. Completes the booking form.
6. Submits the booking.
7. Booking is stored in ServiceNow.
8. Booking Success page is displayed.
9. User can track bookings using My Bookings.
